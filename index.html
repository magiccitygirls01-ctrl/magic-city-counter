Magic City Setup Guide

Step 1: Install AutoHotkey

AutoHotkey is required to run the ticket counter script. Download and install it from the official site:

https://www.autohotkey.com/

Step 2: Download the Ticket Script

Get the Magic City ticket counter script ticket-counter.ahk from the repository or your admin source.

Step 3: Run the Script

  1. Right-click the ticket-counter.ahk file.
  2. Select Run Script.
  3. You should now see the counter start updating when tickets are accepted in-game.

Step 4: Keep the Script Running

Minimize AutoHotkey; do not close it. The counter will only update while the script is active.

Need Help?

Contact your Magic City admin or check the repo for troubleshooting tips.
